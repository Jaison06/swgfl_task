## Installation

Environment : Php(8.x) , composer

git clone <repository-url>

Open the project folder in Command Prompt and run the command below.

composer install

Run the project using PHP's built-in server. You can change the port if needed.

php -S localhost:8080

Alternatively, you can run it using XAMPP or WAMP server.

CSRF token implemented for form security.
